# 这本 RAG 手册覆盖什么与缺什么

## 适合用它学习的内容

- 从文档进入系统，到回答返回用户的端到端流程。
- 文档分块、元数据、Embedding、向量索引和相似度检索的基本关系。
- 查询流转、上下文拼接、提示词、聊天逻辑与异常降级。
- 部署、日志、重索引和常见故障等工程环节。
- 通过 Cloudflare Vectorize + Qwen 看一个具体实现如何落地。

## 不宜只靠本书学习的内容

- BM25 与稠密检索的系统比较。
- 混合检索的融合方法，例如 RRF。
- Cross-Encoder Reranker 的原理和取舍。
- 检索评估：Recall@k、MRR、NDCG 等。
- 生成评估：忠实度、答案相关性、引用正确性。
- 高质量评测集构建和自动回归测试。
- Graph RAG、Agentic RAG、多模态 RAG 的深入实现。
- 权限隔离、提示注入防护和文档级访问控制。

## 学习结论

把本书当成“第一遍端到端主线”和“可运行工程案例”，不要把它当成完整的 RAG 理论教材。读完每个阶段后，用额外资料补齐检索、重排和评估，才能形成能够迁移到其他技术栈的能力。
